# PN532 NFC Payment Skill

This skill helps you work with the PN532 NFC payment project — a C++17 application for
Raspberry Pi that reads NFC cards via libnfc and sends payments via HTTP.

## Project Structure

```
pn532/
├── include/
│   ├── json.h       # nlohmann/json header
│   └── json.hpp
├── src/
│   └── payment.cpp  # Main entry point
├── Makefile
└── README.md
```

## Build & Run

```bash
make          # Build
make run      # Build and run
sudo ./bin/nfc_payment  # Direct execution
```

## Dependencies

- `libnfc-dev` — NFC card reading
- `libcurl4-openssl-dev` — HTTP requests
- nlohmann/json (included)

## Hardware Setup

Connect PN532 via SPI to RPi GPIO pins (see README.md).

## Development

- Add new payment methods in `src/payment.cpp`
- The server endpoint is hardcoded to `http://127.0.0.1:5000/pay`
- UID is sent as JSON with amount field
- Response is parsed for `status` and `remaining` fields
