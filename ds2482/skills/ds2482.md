# ds2482 — Skill de desarrollo

## Lenguajes y herramientas
- C++ (compilador GCC)
- Bash scripting
- I2C (`libi2c-dev`, `i2c-tools`)
- Kernel 1-Wire (`w1` subsystem)

## Estructura
```
ds2482/
├── ds2482Ds1994.cc       Driver principal (I2C)
├── ds1994.cc             Lector alternativo (kernel w1)
├── commandsDs284.sh      Comandos I2C manuales
├── funcCommandsDs1994.sh Comandos I2C con funciones
├── onewired.sh           Inspección de dispositivos 1-Wire
├── compile.sh            Script de compilación rápida
├── .gitignore
├── README.md
└── skills/ds2482.md
```

## Compilación
```bash
g++ -o ds1994_reader ds2482Ds1994.cc -li2c
```

## Notas
- Requiere I2C habilitado en la RPi
- La dirección I2C del DS2482 es `0x18`
- Los scripts Bash usan `i2cset`/`i2cget` de `i2c-tools`
- El lector por kernel w1 necesita el módulo `w1-gpio` y `w1-therm` cargados
