# fflush

## Descripción
Proyecto con utilidades de sistema para forzar el volcado (flush) de búferes de E/S en Linux.

## Dependencias
- libpthread

## Compilación
```bash
ssh joy@raspberry.local "cd /home/joy/src/git/fflush && make"
```

## Binarios
- `bin/flush`: Programa simple de flush
- `bin/app`: Aplicación principal
