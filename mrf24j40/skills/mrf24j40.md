# mrf24j40

## Descripción

Sistema embebido para Raspberry Pi que integra un módulo de radio
MRF24J40MA (ZigBee/802.15.4) con pantalla OLED, comunicación MQTT,
generación de códigos QR y almacenamiento en MySQL.

## Desarrollo

- **Lenguaje:** C++17
- **Compilador:** g++ / clang++
- **Build system:** Makefile (generación automática de reglas)
- **Arquitecturas soportadas:** aarch64, armv7l, x86_64
- **Librerías principales:**
  - `bcm2835` — GPIO y SPI
  - `libmosquitto` — Cliente MQTT
  - `libmysqlcppconn` — Conexión MySQL
  - `libqrencode` — Generación QR
  - `libpng` / `zlib` — Imágenes
  - `SSD1306_OLED_RPI` — Controlador OLED

### Compilar

```bash
make
```

El binario resultante depende de la arquitectura:
- `bin/mrf24_tx_app` (aarch64)
- `bin/mrf24_rx_app` (armv7l)
- `bin/mrf24_app` (x86_64)

### Limpiar

```bash
make clean
make cleanall    # también borra el binario
```

### Compilar librerías por separado

```bash
make libs       # compila mrf24_prj y oled
make oled       # solo OLED
```

### Estructura

```
mrf24_prj/       → Código fuente principal
  app/           → Punto de entrada y lógica de aplicación
  display/       → Controlador OLED SSD1306
  epaper/        → Soporte para displays e-paper
  gpio/          → Manejo de GPIO
  mrf24/         → Driver del MRF24J40MA
  network/       → Cliente MQTT y conexiones de red
  qr/            → Generación de códigos QR
  security/      → Funciones de cifrado
  spi/           → Comunicación SPI
  files/         → Manejo de archivos
  others/        → Utilidades varias
oled/            → Librería OLED independiente
scripts/         → Scripts de instalación y utilidades
install/         → Instaladores de dependencias
```

### Scripts de instalación

```bash
# Todo en uno
sudo bash scripts/dependeciesLibs.sh

# Componentes individuales
sudo bash install/installBCM2835.sh
sudo bash install/installMosquitto.sh
sudo bash install/installOled.sh
sudo bash install/installLibs.sh
```

### Notas

- Requiere `sudo` para acceso a GPIO, SPI y bcm2835
- Configurar Mosquitto en `/etc/mosquitto/mosquitto.conf`
- Habilitar SPI en `raspi-config`
