# ADS1115 — ADC I2C

## Stack técnico

- **Lenguaje:** C++11
- **Build:** CMake + wiringPi
- **Interfaz:** I2C, dirección `0x48`
- **Target:** Raspberry Pi (ARM)

## Comandos útiles

```bash
cd ads1115 && mkdir build && cd build && cmake .. && make
sudo ./ads1115_app
```

## Cableado entradas analógicas

Usa divisores R1=10kΩ / R2=1kΩ con ganancia `GAIN_EIGHT` (±0.512V). Fórmula:

```
V_in = ADC_valor * (rango_ganancia / 32767.0) * (R1 + R2) / R2
```
