# 25AA02E48 — EEPROM SPI

## Stack técnico

- **Lenguaje:** C++17
- **Build:** CMake
- **Interfaz:** SPI (`/dev/spidev0.0`, modo 0, 1 MHz)
- **Target:** Raspberry Pi (ARM)

## Comandos útiles

```bash
mkdir build && cd build && cmake .. && make
sudo ./build/app_25aa
```

## IOCTL SPI

El driver usa `SPI_IOC_MESSAGE` para transferencias full-duplex. Ver `main.cc:89`.

## Operaciones soportadas

- `READ`  (0x03) — lectura de memoria
- `WRITE` (0x02) — escritura de memoria
- `RDSR`  (0x05) — leer status register
- `WRSR`  (0x01) — escribir status register
- `WREN`  (0x06) — habilitar escritura
- `WRDI`  (0x04) — deshabilitar escritura
