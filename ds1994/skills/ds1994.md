# DS1994-F5 — iButton 1-Wire

## Stack técnico

- **Lenguaje:** C++17 (driver) + Bash (scripts)
- **Build:** Makefile / g++ directo
- **Interfaz:** 1-Wire, `/sys/devices/w1_bus_master1/<id>/rw`
- **Target:** Raspberry Pi (ARM)

## Comandos útiles

```bash
# Driver C++
cd src_deepseek
g++ -std=c++17 -o ds1994_app main.cpp -pthread
sudo ./ds1994_app info

# Proyecto completo con Makefile
cd ds1994_deepseek_success/ds1994_project_fixed
make && sudo ./bin/ds1994_app info

# Scripts de diagnóstico
sudo bash scrpits_ds1994/busqueda_ds1994.sh
sudo bash scrpits_ds1994/reset_kernel_w1_ds1994.sh
```

## Registros de memoria

| Dirección | Registro      | Bytes |
|-----------|---------------|-------|
| 0x0200    | Status        | 1     |
| 0x0201    | Control       | 1     |
| 0x0202    | RTC           | 5     |
| 0x0207    | Interval Timer| 5     |
| 0x020C    | Cycle Counter | 4     |

## Notas de desarrollo

- SIN usar SKIP_ROM — sigue estrictamente el datasheet
