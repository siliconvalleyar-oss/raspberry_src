# ALS31313_Pi — Sensor Hall 3D

## Stack técnico

- **Lenguaje:** C11
- **Build:** CMake
- **Interfaz:** I2C (`/dev/i2c-1`)
- **Target:** Raspberry Pi / SBC Linux

## Comandos útiles

```bash
mkdir build && cd build && cmake .. && make
./als31313 -a 0x60 -b 1 -d 200
```

## Opciones CLI (argp)

El programa usa `argp` para parsing de argumentos:
- `-b` bus, `-a` dirección, `-d` delay, `-f` filtro, `-r` resolución, `-q` quiet

## Notas de desarrollo

- `src/als31313.c` — archivo único con todo el driver
- Soporta temperatura ambiente y campos magnéticos X/Y/Z
- incluye `initHardware()`, `readAmbientTemp()`, `readHall()`
